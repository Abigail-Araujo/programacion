¿Qué son los lenguajes de programación?

Los lenguajes de programación están compuestos por términos y comandos específicos, para crear cosas como páginas web, aplicaciones y básicamente cualquier software.

Lenguajes como Javascript, Python, son usados frecuentemente en sitios web. C++ es empleado en muchos lugares para hacer cosas como programas de escritorio, juegos, servidores web, entre otros. También existen HTML y CSS aunque estos tecnicamnete no son lenguajes de programación, son usados para crear la estructura.




Elementos de un lenguaje de programación

Tipos de datos

Enteros

Sirven para representar números enteros.

Reales

Sirven para representar números con punto decimal.

Cadena

Sirven para representar datos de tipo texto o carácter.


Palabras reservadas

Se trata de un conjunto de palabras que el lenguaje de programación considera propio de su sintaxis y son empleados para construir instrucciones.


Operadores

Son usados para crear instrucciones realizando cálculos matemáticos, comparación y operaciones lógicas.

Operadores aritméticos

  • Exponenciación.
  • Producto y división.
  • Suma y resta.

Operadores relacionales

  • Mayor: >
  • Mayor o igual: >=
  • Menor: <
  • Menor o igual: <=
  • Igual: ==
  • Distinto: <>

Operadores lógicos

  • Y: and
  • O: or
  • No: not

Constantes y variables

Las constantes son datos que no cambian durante la ejecución del programa, por el contrario las variables son datos que pueden cambiar durante el proceso de ejecución.


Sintaxis

Es el conjunto de reglas que deben seguirse al escribir el código fuente de los programas, para considerarse como correcto en ese lenguaje de programación.




Otros conceptos importantes

Código fuente

Es un conjunto de líneas de texto, que son las instrucciones que debe seguir la computadora para ejecutar dicho programa.


Número binario

El código binario es el sistema númerico usado para la representación de texto o procesadores de instrucciones de computadores utilizando el sistema binario.


Sistema binario

Es un sistema de numeración en el que los números se representan solo usando las cifras 0 y 1.




Paradigmas de programación

Programación estructurada

Es un paradigma secuencial, las instrucciones van de arriba hacia abajo, tiene la posibilidad de estructurar el programa en piezas pequeñas como son las funciones. Posee 3 estructuras básicas: secuencial, selectiva, repetitiva. Tiene un único punto de entrada y sólo uno de salida.


Programación modular

Incluye a la estructurada pero divide los programas en módulos más sencillos llamados subprogramas. El programa principal coordina las llamadas a los módulos secundarios y pasa los datos necesarios en forma de parámetros. A su vez cada módulo puede contener sus propios datos y llamar a otros módulos o funciones.


Programación orientada a objetos

Es un paradigma que construye modelos de objetos que representan elementos del problema. Crea estructuras llamadas clases que permiten agrupar funcionalidades. Además permite la reutilización de código. Las clases pueden crear objetos con ciertos atributos y métodos.
